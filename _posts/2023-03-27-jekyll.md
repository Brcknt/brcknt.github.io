This website was created using a static website generator called Jekyll

## Jekyll

Jekyll is a tool for generating static websites from a number of source files. it means that I don't have to manually code in everything I need every single time. Insteead, i create a single HTML document as a "layout", and any number of markdown documents containing the actual content I want, and Jeklyll takes and combines these automatically into a set of ready-made pages.

This is especially useful to me because it leaves me able to go back and change things later, even after there is already content on the page
